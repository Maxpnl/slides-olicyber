# HTML, CSS e Javascript
<!-- fragments-start -->
HTML è un linguaggio di markup, ha una struttura ("a matrioska"), serve a definire COSA vediamo

CSS è un linguaggio di stile, serve a definire COME vediamo ciò che vediamo, quindi colori, posizioni, etc degli elementi

JS è un linguaggio di programmazione, serve a rendere la pagina dinamica
<!-- fragments-end -->

---

## Esempio di una struttura HTML

import { ExampleStruttura } from '../scripts/Examples';

<ExampleStruttura />

---

## Attributi

I tag HTML possono avere attributi, del tipo `nome="valore"`, ad esempio
```html
<div id="il-mio-id" class="la-mia-classe" nome="valore"></div>
```

## Alcuni tag importanti
<!-- fragments-start -->
```html
<!-- Base -->
<html>
    <head>
        <title>Questo è il titolo della scheda</title>
        <meta charset="UTF-8">
    </head>
    <body>
        <div>
        </div>
    </body>
</html>
```
```html
<!-- Tipografia -->
<h1>Titolo</h1>
<h2>Titolo piccolo</h2>
<h3>Titoletto</h3>
<h4>Titolettino</h4>
<h5>Titoluccio</h5>
<h6>Titoluccino</h6>
<p>Questo è un paragrafo</p>
<br>
```
```html
<!-- Inviare richieste -->
<form action="/endpoint/a/cui/mandare/i/dati" method="POST">
  <input type="text" id="nome" name="nome">
  <input type="password" id="password" name="password">
  <input type="checkbox" id="ricordami" name="ricordami" value="ricordami">
  <input type="submit" value="Invia form">
</form> 
```
```html
<!-- Link, bottoni, immagini, iframe -->
<a href="https://google.it" target="_blank">Clicca qui per aprire google!</a>
<button>Cliccami!</button>
<img src="/immagine.jpg" />
<iframe src="https://google.com"></iframe>
```
```html
<!-- Caricare script (Javascript) e codice di stile (CSS) -->
<script src="/miofile.js"></script>
<script>alert(1)</script>

<link rel="stylesheet" href="/stile.css">
<style>
#test {
    color: blue;
}
</style>
<!-- (# -> id, . -> class) -->
```
